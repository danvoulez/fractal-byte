# Governance (v0)

- **Canon** técnico: NRF‑1.1 é a base imutável para bytes e CID.
- **Decisões** são publicadas como D# em DECISIONS.md (ex.: D8 – deterministic binding).
- **Chaves** e **receipts**: rotação versionada; auditável.
- **Compatibilidade**: alterações compatíveis sempre que possível; breaking requer janela e migração.
